# Subnet Calculator v1.0
**Author:** Jonathan Cardoso  
**Language:** Python 3  

## Description
A command-line subnet calculator built in Python that takes an IP address 
and CIDR notation as input and returns key subnet information instantly.

## Features
- Network Address
- Subnet Mask
- Broadcast Address
- Usable Host Range
- Total Usable Hosts
